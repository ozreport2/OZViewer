# 입력 컴포넌트가 있는 보고서에서 틀 고정 동작 확인


Feature: 보고서 틀 고정 기본 동작 확인 - 입력 컴포넌트 있음


    Background:
        Given 필수파일 및 뷰어패러미터 설정
            | 보고서파일   | ozr/FreezePane_TopBottom.ozr, ozr/FreezePane_Top.ozr   |
            | 뷰어패러미터1   | 보고서실행   |
            | 뷰어패러미터2   | viewer.pagedisplay=continuous   |


    Scenario Outline: 1.
        Given 뷰어패러미터1 실행
        And 세로방향 스크롤이 표시된 상태
        When <PageNo>로 이동
        And 스크롤을 위/아래 끝까지 최소 4번 이동
        Then 페이지에 있는 빨간색 점선이 가져지지 않고 고정되어 표시됨 # 빨간색 점선 위치는 보고서 화면에서 확인
        Examples:
            | PageNo   |
            | 1   |
            | 2   |
            | 3   |
            | 4   |


    Scenario: 2.
        Given 뷰어패러미터2 실행
        When 스크롤을 위/아래 끝까지 최소 4번 이동
        Then 페이지에 있는 빨간색 점선이 가져지지 않고 고정되어 표시됨 # 빨간색 점선 위치는 보고서 화면에서 확인
